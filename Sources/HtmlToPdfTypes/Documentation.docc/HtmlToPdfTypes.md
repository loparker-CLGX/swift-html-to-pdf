# ``HtmlToPdfTypes``

Core type definitions for PDF generation.

## Overview

HtmlToPdfTypes defines all the fundamental types, protocols, and enums used throughout swift-html-to-pdf. These types provide the foundation for type-safe PDF generation with full Swift 6 concurrency support.

### Core Types

- ``PDF``
- ``PDF/Document``
- ``PDF/Configuration``
- ``PDF/Render``
- ``PDF/Render/Result``
- ``PDF/Render/Client``
